Music & Mental Health Analysis
OMIS 3020 - Group Project
This project analyzes the relationship between music listening habits and self-reported mental health outcomes using a publicly available survey dataset of 736 respondents.
Research Questions

Is there a measurable association between daily listening hours and depression scores?
Are specific genres associated with elevated anxiety or insomnia?
Does BPM correlate with insomnia?
Which variable best predicts whether music improves or worsens mood?

Dataset
Music & Mental Health Survey Results - available on Kaggle
How to Run
No installation required. This project runs entirely in Google Colab.

Open Final_Code.ipynb in Google Colab
Upload mxmh_survey_results.csv using the Files sidebar
Select Runtime > Run All

All 12 visualizations will render inline. Statistical results print below each cell.
Requirements
All libraries are pre-installed in Google Colab: pandas, numpy, matplotlib, seaborn, scikit-learn, scipy
